EmailJS Setup Guide

EmailJS sends email directly from your browser. Your CV data never touches any server you don't own. Credentials are only stored in this browser session.

Step-by-step: get your free credentials

1
Create a free account at emailjs.com — no credit card needed. Free plan gives 200 emails/month.
2
In your dashboard → Email Services → Add New Service. Connect Gmail, Outlook, or any SMTP you own. After connecting, copy the Service ID shown (e.g. service_abc1234).
3
Go to Email Templates → Create New Template. In the template body, use these variables:

{{to_name}} — recipient's full name
{{cv_full_text}} — full plain-text CV summary
{{personal_note}} — personal note (if any)

Set the To Email field to {{to_email}}.
Set Reply To to {{reply_to}}.
Suggested Subject: Your CV Summary — {{to_name}}

Copy the Template ID (e.g. template_xyz9876).
4
Go to Account → General → copy your Public Key (looks like a random 20-char string).
5
Paste all three values in the fields below and click Save Credentials. Then use the Send Email Copy button above.
Dashboard → Account → General → Public Key
